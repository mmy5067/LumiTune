# Privacy Notice

## Summary

LumiTune is a local desktop wrapper around Apple Music Web.
It does not operate its own cloud backend, user account system, or analytics service.

## What LumiTune stores locally

LumiTune stores a small amount of local preference data on your device so the overlay can reopen the way you left it.

This may include:

- Overlay position
- Overlay size
- Overlay opacity
- Overlay UI preferences

These values are stored locally on your machine and are not sent to a LumiTune server.

## What LumiTune reads locally

To show now playing information in the floating mini player, LumiTune reads media session metadata exposed by Windows.

This may include:

- Track title
- Artist name
- Album name
- Playback status

This information is read locally for runtime display and playback controls.

## Network requests

LumiTune loads Apple Music Web directly in the main window.

That means the app will connect to Apple services used by:

- `https://music.apple.com`
- related Apple Music Web resources needed for playback and sign-in

For album artwork in the floating mini player, LumiTune may also request artwork metadata from Apple's public iTunes Search endpoint.

## What LumiTune does not do

LumiTune does not:

- run its own telemetry or analytics backend
- upload your listening history to a LumiTune server
- collect your Apple account credentials
- download protected Apple Music media files
- bypass DRM

## Third-party services

Because LumiTune wraps Apple Music Web, your use of Apple Music inside the app is also subject to Apple's own terms and privacy practices.

## Contact and changes

If this privacy notice changes in the future, the repository README and this file should be updated together.
