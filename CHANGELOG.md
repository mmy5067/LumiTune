# Changelog

All notable changes to LumiTune will be documented in this file.

## [0.1.0] - 2026-05-19

First public Windows preview.

### Added

- Apple Music Web main window powered by Tauri
- System tray controls for show, hide, and playback actions
- Floating mini player with drag, hide, opacity, and size controls
- Persistent overlay position and sizing
- Now playing metadata in the overlay
- Album artwork support in the overlay
- Updated application branding and icons

### Notes

- Windows only
- No lyrics support
- No MusicKit integration
- No Apple Music API integration
